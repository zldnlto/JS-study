# 값식문

값 : 하나의 값  
식 or 표현식 (expression) : 값을 생성, 값으로 귀결  
문 (statement) : 행동을 한다. 흐름을 제어한다. 인터프리터에게 명령한다. 그것을 지시문으로  
문을 사용하면서부터 프로그래밍 언어다.

```js
//값
1("문자열 값");
true;
null;
undefined;
//값으로 귀결된 식 or 연산식
1 + 1;
1 > 2;
//문
if (식) {
  return true;
}
for(식){
    ..
}
while(식) {

}
```

##### 의식적으로 판단하며 코딩하기!

```js
for (반복 조건 초기화) {
    if (기상 상태는 비가온다) {
        //우산을 쓴다
    }else if {
        //우산을 챙긴다
    }
}
```
